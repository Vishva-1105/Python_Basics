# PythonProject

This repository contains my Python practice projects, including scripts, CSV files, and text files.  
It covers basic Python concepts as well as Object-Oriented Programming and data handling with NumPy and Pandas.

---

## Files Included

### Python Scripts
- `OOP.py` — Object-Oriented Programming examples  
- `fundamentals.py` — Python basics  
- `main.py` — Main script for running projects  
- `test.py` — Testing scripts  
- `myModule.py` — Custom module for reuse  
- `text_numpy.py` — NumPy examples  
- `text_pandas.py` — Pandas examples  

### Data Files

- `students.csv` — Student dataset  
- `data.txt` — Text data  

### Others
- `requirements.txt` — Python dependencies
